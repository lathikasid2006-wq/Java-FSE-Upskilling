Exercise 1 · Setting Up Bootstrap 5

1.1 – Link Bootstrap 5 via CDN

Paste the following tags inside <head> (CSS) and before </body> (JS):

<!-- CSS -->
<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css">

<!-- JS bundle (includes Popper) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

Result – this button uses Bootstrap from the CDN above:

1.2 – Using Downloaded Bootstrap Files

After downloading from getbootstrap.com, place the files in your project and reference them locally:

<link rel="stylesheet" href="assets/css/bootstrap.min.css">
<script src="assets/js/bootstrap.bundle.min.js"></script>

Install via npm: npm install bootstrap

Exercise 2 · Bootstrap Structure & Files

2.1 – Bootstrap Directory Structure

bootstrap/
├── css/
│   ├── bootstrap.min.css        ← compiled, minified stylesheet
│   ├── bootstrap.css            ← full (readable) stylesheet
│   ├── bootstrap-grid.css       ← grid-only build
│   ├── bootstrap-reboot.css     ← CSS reset only
│   └── bootstrap-utilities.css  ← utility classes only
├── js/
│   ├── bootstrap.bundle.min.js  ← JS + Popper.js bundled
│   ├── bootstrap.min.js         ← JS only (Popper separate)
│   └── bootstrap.esm.min.js     ← ES Module version
└── icons/                       ← (separate Bootstrap Icons package)
    └── bootstrap-icons.css

2.2 – Including bootstrap.bundle.min.js

<!-- Before </body> -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

The bundle version includes Popper.js, required for dropdowns, tooltips, and popovers.

Exercise 19 · Customisation with Sass

19.1 – Set Up Bootstrap with Sass

# 1. Create project & install
mkdir my-project && cd my-project
npm init -y
npm install bootstrap sass

# 2. Create src/custom.scss
# 3. Compile
npx sass src/custom.scss dist/custom.css --watch

19.2 – Customise via _variables.scss

$primary:        #e63946;          
$border-radius:  1rem;           
$font-family-base: 'Poppins', sans-serif;
@import "../node_modules/bootstrap/scss/bootstrap";
.hero { background: $primary; color: #fff; padding: 4rem; }
Key rule: Always declare your variable overrides before @import "bootstrap". Bootstrap uses !default on all variables, so your values take precedence only when declared first.
